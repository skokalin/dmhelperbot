# ✨ Предложения по развитию функционала бота (Upcoming Updates)

Ниже представлен список идей для развития текущей функциональности бота. Ничего в коде не изменено.

## 🗓️ Управление событиями (Calendar Management)

*   **Изменение или отмена события по ID:** Добавить явную возможность модифицировать или отменять событие, используя его уникальный ID.
*   **Повторяющиеся события:** Реализовать возможность при добавлении события указать, что оно должно повторяться (например, "каждый понедельник", "ежемесячно 1-е число").
*   **Фильтр по дате:** Расширить команду `/list`, чтобы она могла принимать аргументы для отображения событий только за заданную дату или в заданном диапазоне.

## 📢 Управление рассылками (Broadcast/Chat Management)

*   **Отправка тестового сообщения:** Реализовать команду (например, `/test_broadcast`), которая позволяет отправить тестовое сообщение всем сохраненным чатам для проверки работоспособности рассылки.
*   **Удаление чатов:** Добавить команду (например, `/remove_chat <chat_id>`), чтобы пользователи могли управлять списком сохраненных чатов.
*   **Управление группами/админами:** Внедрить механику тегирования чатов (например, "Группа", "Канал") для лучшей категоризации в списке.

## ✨ Пользовательский опыт и утилиты (UX & Utility)

*   **Быстрая статистика:** Добавить команду `/stats`, которая будет выводить общую полезную статистику: общее количество запланированных событий и количество сохраненных чатов.
*   **Улучшенная справка/Помощь:** Расширить функциональность команды `/help` (или `/start`), добавив подробное описание синтаксиса и ожидаемого ввода для всех основных команд.
